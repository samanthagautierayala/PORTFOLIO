THE WILD DOLPHIN PROJECT REBRANDING

I designed a rebranding for a non-profit organization called The Wild Dolphin Project. For this project, I created a comprehensive and cohesive brand guideline book, which includes key visual assets and also provides design elements the client was in need of. The following brand book contains: brand overview, a description of the new logo, logo usage and variations, typography, color palette, patterns, a stationary system, promotional items and marketing collateral.
This project was made as an assignment for the UM course 'STC202 - Advanced Graphic Design for Advertising.'
